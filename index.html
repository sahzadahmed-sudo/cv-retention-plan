# College Vidya Retention Plan ## Overview The College Vidya Retention Plan is designed to improve student success through various innovative features and tools. This application includes: - **Retention Flowchart**: An interactive flowchart that guides students through the retention process, with nodes that provide detailed information when clicked. - **AI-Powered Call Script Generator**: This feature integrates with Claude to generate call scripts tailored for various student scenarios, enhancing communication effectiveness. - **University Payout Table**: A comprehensive payout table that allows users to import/export data using Excel, ensuring smooth data management. - **Subsidy Policy Overview**: A detailed overview of the subsidy policies available, helping students understand financial support options. - **Professional Styling**: The application features modern professional styling with color-coded nodes for easy navigation and understanding. - **Local Storage Persistence**: University data is stored locally, ensuring quick access and reliability across sessions. - **Modal Dialogs**: Interactive modal dialogs enhance user experience by providing context-sensitive information and options for user interactions. - **JavaScript Functionality**: All necessary JavaScript functionality is included to ensure a smooth and interactive experience for all users. ## Conclusion The College Vidya Retention Plan application aims to streamline communication and support for students, making the retention process more efficient and effective. ---
